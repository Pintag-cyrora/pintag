Some words are left in English because that reflects how they're actually used in natural, modern Lao — translating them would itself be the unnatural choice, not the correct one. Never translate these unless explicitly instructed otherwise for a specific piece.

**Starting list (founder-specified):**
- AI
- TikTok
- Facebook
- Instagram
- Focus
- Stress
- Burnout
- Memory
- Dopamine
- Active Recall
- Passive
- Multitask
- Podcast

**This is a seed list, not a closed one.** Marked `verified` because it's the founder's own direct specification, not because it's exhaustive. New candidates for this list should be captured the same way any other reusable insight is — via `proposeSuggestion()` (`kind: 'wording-improvement'`, `suggestedCategory: 'language/terminology'`) when one is genuinely observed in real usage during content review, then reviewed and merged into this entry (or a successor) rather than invented speculatively from outside real Lao usage.

**Relationship to `brain/lao/dictionary.md`:** this list is about *which words don't get translated at all*; the dictionary is about *how Lao terms that do get used are defined and applied*. Both are canonical, for different questions — check both before writing.
