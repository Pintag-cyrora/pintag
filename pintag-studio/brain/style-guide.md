# Style Guide

*Mechanical/formatting companion to `brain/brand-voice.md` (tone) and `brain/posting-rules.md` (platform/legal rules).*

## Formatting

- Sentence case for headlines, not title case.
- Numbers: use figures for prices, measurements, and statistics (e.g. "3 bedrooms," "120 sqm," "500,000,000 LAK") even below ten, since these are data points readers scan for.
- District and village names: match `knowledge-base/neighborhoods/` spelling exactly, every time, across all languages.
- Currency: always state LAK (and USD equivalent where helpful) explicitly — never an unlabeled number.

## Language-Specific Notes

- **Lao (lo):** default language for all consumer-facing content unless a piece is explicitly scoped English- or Chinese-first.
- **English (en):** plain, international real-estate English — avoid US/UK-specific idioms that don't translate to an international buyer audience.
- **Chinese (zh):** Simplified Chinese, written for investor/buyer audiences already common in the Pintag audience per `knowledge-base/company/audience-personas.md`.
- Trilingual parity is a Year-1+ goal (see Writer's future improvements), not a hard requirement for every single post from day one — Brand Guardian checks voice consistency within whichever language(s) a piece is published in.

## Readability

- Target a reading level accessible to a general adult audience in each language — no unexplained jargon on first use (define financing/legal terms plainly, once, per piece).
- Prefer short paragraphs (2–3 sentences) for social formats; longer guides may use subheadings but should still avoid dense blocks of text.

## Grammar & Proofreading

- Brand Guardian checks grammar/readability as part of its review pass (see `.claude/agents/brand-guardian.md`) — Writer should not rely on Guardian to catch avoidable errors; treat this as a self-check before handoff too.
